Introducción a Arduino en Windows

Obtener una placa Arduino y el cable USB

En este manual, suponemos que estás utilizando un Arduino Uno, Arduino Duemilanove, Nano, Arduino Mega 2560, o Diecimila.También se necesita un cable USB estándar (A para clavija B): el tipo que se conecta a una impresora USB, por ejemplo. (Para el Arduino Nano, necesitarás un cable de A a mini-B en su lugar.).

Descarga el IDE de Arduino

Obtén la última versión estable desde la página de descargas.

Después de que el archivo sea descargado, da clic sobre el ejecutable y realiza la instalación.

Conecte la placa

Conectar el Arduino Uno, Mega, Duemilanove o Arduino Nano con el cable USB a la computadora o con una fuente externa de poder.Si usted está utilizando un Arduino Diecimila, usted necesita asegurarse de que la tarjeta está configurada para extraer energía de la conexión USB. La fuente de alimentación se selecciona con un puente, colocando una pequeña pieza de plástico entre dos de los tres pasadores (“pines”) entre las tomas USB y de alimentación. Compruebe que está en los dos pines más cercanos al puerto USB.

Conecte la placa Arduino al computador mediante el cable USB. El LED verde de energía (etiquetado como PWR) debe encenderse.

Instale los controladores

Instalación de Controladores para el Arduino Uno o Arduino Mega 2560 con Windows 7, Vista o XP:

  • Enchufe su tablero y esperar para que Windows iniciar su proceso de instalación del driver. Después de unos momentos, el proceso va a fallar, a pesar de sus esfuerzos.
  • De clic en Inicio ---> Panel de Control--->Sistema y Seguridad ---> Sistema-->Administrador de dispositivos. Como se muestra en la imagen.

  • Despues diríjase a la categoría Puertos(COM & LPT), donde encontrarás un puerto llamado Arduino Uno (COMxx), de clic derecho, elige la opción “Actualizar Driver” y a continuación en “Buscar Software del Driver en mi computadora” y busque el archivo llamado arduino.inf en la carpeta de instalacion de Arduino Uno:

          “C:\Program Files (x86)\Arduino\drivers”

Inicie la aplicación Arduino

Haga doble clic en la aplicación Arduino. (Nota: si se carga el software de Arduino en el idioma equivocado, puedes cambiarlo en el diálogo de preferencias)

Practica 1: Blink

Material

  • 1 Protoboard
  • 1 Arduino UNO
  • Alambre para protoboard
  • Conector USB
  • 1 LED

Circuito

Codigo

int led = 13; // El pin que usaremos es el 13 y lo guardamos en la variable led.

void setup() {                

  // inicializa el pin digital como una salida.

  pinMode(led, OUTPUT);    

}

// el ciclo no para de correr:

void loop() {

  digitalWrite(led, HIGH);   // enciende el LED (HIGH es el nivel de voltaje)

  delay(1000);               // espera un segundo

  digitalWrite(led, LOW);    //  apaga el LED poniendo el voltaje en LOW

  delay(1000);               // espera un segundo

}

Pasos

Primeramente identificaremos los pines del LED ya que tiene un positivo y un negativo, la patita mas larga del LED es el positivo y la más corta es el negativo.

Despues coloca tu LED en el protoboard y conecta el positivo del LED al PIN 13 de tu Arduino

Ahora lo que resta es conectar de la misma manera el PIN restante del LED que es el negativo al GND del Arduino.

Práctica 2: Sweep

Materiales

  • Arduino UNO
  • Servo Motor
  • Cable

Circuito

Código

#include <Servo.h>

Servo myservo;  // Crea un objeto tipo Servo. Un máximo de 8 objetos tipo servo pueden ser creados.

int pos = 0;    // variable que guarda posición del servo

 

void setup()

{

  myservo.attach(9);  // Se asigna el pin 9 al servo.

}

void loop()

{

  for(pos = 0; pos < 180; pos += 1)  // Va desde 0 a 180 grados, en pasos de un grado.

  {                                  

    myservo.write(pos);              // El servo va a la posición que tiene la variable 'pos'

    delay(15);                       // Espera al servo 15ms para que llegue a la posición

  }

  for(pos = 180; pos>=1; pos-=1)     // Va desde 180 a 0 grados

  {                                

    myservo.write(pos);              // El servo va a la posición que tiene la variable 'pos'

    delay(15);                       // Espera al servo 15ms para que llegue a la posición

}

}

Conecta el otro extremo del alambre a voltaje en tu protoboard

Conecta otro alambre en el pin de GND

Conecta el otro extremo del segundo alambre a negativo de tu protoboard.

Conecta tres alambres de la siguiente manera: Amarillo con Amarillo, Rojo con Rojo y Café con Negro. Recuerda que la configuración de color de tu servo puede ser diferente a la mostrada.

Conecta el extremo del alambre que va conectado al servo en el pin 9 (Digital)

Ahora conecta el alambre color rojo que está en el servo, al corriente del protoboard

Código

Arduino

Este código puede ser adaptado para utilizar dos brazos, pero necesitarás un par de servos más.

#include <Servo.h>

/* Declaramos los servomotores. Solo utilizaremos dos de ellos, pero puedes aplicarlo a 4*/

Servo leftshoulder;

Servo leftelbow;

Servo rightshoulder;

Servo rightelbow;

// Utilizaremos un arreglo para las 2 posiciones del servo.

int nextServo = 0;

int servoAngles[] = {0};

void setup() {

 // Conecta el pin 3 de los servos

 leftshoulder.attach(9);

 leftelbow.attach(10);

 rightshoulder.attach(11);

  rightelbow.attach(3);

 Serial.begin(9600);

}

void loop() {

 if(Serial.available()){

 int servoAngle = Serial.read();

 servoAngles[nextServo] = servoAngle;

 nextServo++;

 if(nextServo >0){

 nextServo = 0;

 }

 leftshoulder.write(servoAngles[0]);

 }

}

Processing

A continuación copia y pega el siguiente código en el programa processing

import SimpleOpenNI.*;

SimpleOpenNI kinect;

import processing.serial.*;

Serial port;

void setup() {

 size(640, 480);

 kinect = new SimpleOpenNI(this);

 kinect.enableDepth();

 kinect.enableUser();

 kinect.setMirror(true);

 println(Serial.list());

 String portName = Serial.list()[0];

 port = new Serial(this, portName, 9600);

}

void draw() {

 kinect.update();

 PImage depth = kinect.depthImage();

 image(depth, 0, 0);

 IntVector userList = new IntVector();

 kinect.getUsers(userList);

 if (userList.size() > 0) {

   int userId = userList.get(0);

   if ( kinect.isTrackingSkeleton(userId)) {

     // get the positions of the three joints of our arm

     PVector rightHand = new PVector();

     kinect.getJointPositionSkeleton(userId,

       SimpleOpenNI.SKEL_RIGHT_HAND,

       rightHand);

     PVector rightElbow = new PVector();

     kinect.getJointPositionSkeleton(userId,

       SimpleOpenNI.SKEL_RIGHT_ELBOW,

       rightElbow);

     PVector rightShoulder = new PVector();

     kinect.getJointPositionSkeleton(userId,

       SimpleOpenNI.SKEL_RIGHT_SHOULDER,

       rightShoulder);

           // we need right hip to orient the shoulder angle

     PVector rightHip = new PVector();

     kinect.getJointPositionSkeleton(userId,

       SimpleOpenNI.SKEL_RIGHT_HIP,

       rightHip);

           // reduce our joint vectors to two dimensions

     PVector rightHand2D = new PVector(rightHand.x, rightHand.y);

     PVector rightElbow2D = new PVector(rightElbow.x, rightElbow.y);

     PVector rightShoulder2D = new PVector(rightShoulder.x,

       rightShoulder.y);

     PVector rightHip2D = new PVector(rightHip.x, rightHip.y);

           // calculate the axes against which we want to measure our angles

     PVector torsoOrientation =

     PVector.sub(rightShoulder2D, rightHip2D);

     PVector upperArmOrientation =

     PVector.sub(rightElbow2D, rightShoulder2D);

           // calculate the angles between our joints

     float shoulderAngle = angleOf(rightElbow2D,

       rightShoulder2D,

       torsoOrientation);

     float elbowAngle = angleOf(rightHand2D,

       rightElbow2D,

       upperArmOrientation);

           // show the angles on the screen for debugging

     fill(255,0,0);

     scale(3);

     text("shoulder: " + int(shoulderAngle) + "\n" +

       " elbow: " + int(elbowAngle), 20, 20);

     byte out[] = new byte[2];

     out[0] = byte(int(shoulderAngle));

     out[1] = byte(int(elbowAngle));

     port.write(out);

   }

 }

}

float angleOf(PVector one, PVector two, PVector axis) {

 PVector limb = PVector.sub(two, one);

 return degrees(PVector.angleBetween(limb, axis));

}
